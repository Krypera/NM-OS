# Installing New Software

NM-OS is ready for work right out of the box. But if you need more tools, finding and installing them is safe and straightforward.

## The Software App
The safest and easiest way to install new applications is using the **Software** application.
You can find it by pressing the Super key and typing "Software".

1. Open the **Software** app.
2. Use the search bar at the top to find the app you want (for example, "Spotify", "VLC", or "GIMP").
3. Click the application, then click **Install**. 
4. The application will be downloaded and installed automatically. 

All applications installed through the Software app use "Flatpak" technology. This means they run inside a secure container (sandbox) and cannot mess with your core operating system files.

## Managing Permissions
When an application needs to access your files, your camera, or your microphone, it will ask you for permission the first time it tries to use them.

You can manage these permissions at any time:
1. Open the System settings (Settings app).
2. Go to the **Apps** section.
3. Select an application to see and change what it is allowed to do.

## System Updates
When updates are available for your operating system or your applications, you will receive a notification. You can install all updates simultaneously from the **Updates** tab inside the **Software** app. Keeping your system updated is the easiest way to stay secure.
